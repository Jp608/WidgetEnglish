# WidgetEnglish
WidgetEnglish es una aplicación móvil de aprendizaje de inglés basada en micro interacciones y widgets interactivos, que integra principios de repetición espaciada, aprendizaje adaptativo e inteligencia artificial para ofrecer una experiencia de aprendizaje personalizada, continua y distribuida a lo largo de la rutina diaria del usuario.
